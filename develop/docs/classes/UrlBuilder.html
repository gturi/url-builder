addPath

  • addPath(path: string, trailingSeparator?: boolean): UrlBuilder
    • Defined in url-builder.ts:37

    Adds a path to the current url. The resulting url is obtained by appending to the base one '/' separator followed by the specified path.

    Parameters

    • path: string

      the path that should be appended to the current url.

    • trailingSeparator: boolean = false

      if set to true adds a trailing '/' after the specified path. This property should be used only for compatibility reasons, since trailing '/' should be avoided when building rest APIs.

    Returns UrlBuilder

    a new UrlBuilder instance.

addPathVariable

  • addPathVariable(pathVariable: string, trim?: boolean, trailingSeparator?: boolean): UrlBuilder
    • Defined in url-builder.ts:62

    Adds a path variable to the current url. The resulting url is obtained by appending to the base one '/' separator followed by the specified pathVariable. Path variables can contain spaces and any '/' character will be replaced with '%2F'.

    Parameters

    • pathVariable: string

      the path variable that should be appended to the current url.

    • trim: boolean = false

      whether the path variable should be trimmed.

    • trailingSeparator: boolean = false

      if set to true adds a trailing '/' after the specified pathVariable. This property should be used only for compatibility reasons, since trailing '/' should be avoided when building rest APIs.

    Returns UrlBuilder

    a new UrlBuilder instance.

    Adds a query param to the url. The resulting url is obtained by appending to the base one '?' or '&' followed by key=value string. '?' separator will be used the first time this function gets called successfully, '&' separator will be used the other times.

    Parameters

    • key: string

      the query param name.

    • value: string | number | boolean | object

      the value of the query param. Values with type == object will converted to JSON string.

    Returns UrlParamBuilder

    a new {@link UrlParamBuilder} instance.
